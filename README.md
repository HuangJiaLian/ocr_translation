# OCR Translation
## 说明
使用基于`LSTM`的`Tesseract 4.0`来识别你选中的区域的英文单词或者句子，对其进行翻译。
翻译结果通过通知弹出，同时将结果放到鼠标中键缓冲区。将你选择的图片区域放在剪贴板缓冲区。

## 添加到菜单
```
[Desktop Entry]
Encoding=UTF-8
Exec=/home/jack/.jackprograms/bin/python /home/jack/jack/Documents/github/OCR_Translation/ocr_translation.py

StartupNotify=true
```
